一、web前端做什么？
1.提供交互界面
2.编写交互逻辑
3.数据展示
二、web前端需要学什么？
1.html：编写网页内容
2.css：布局（美化网页内容）
3.JavaScript：网页的交互（行为）
特点：弱类型语言、非常灵活、脚本语言、面向对象、自上而下解释执行

alert：警告框
在界面中ctrl shift+i=F12
console：控制
document 文档
uncaught 未捕获 invalid 无效的

三、js基础
1.js语法规范
①、每一个js语句完了以后需要用英文的分号结尾
②、js严格区分大小写
2.输出
①、alert(“警告框中输出”)；
②、console.log(“控制台中输出”)；
③、document.write(“页面中输出”)；
3.变量：内存中存储数据的一片空间
①、变量的命名规则：
必须以英文字母、$、 _ 开头，后面可以拼接数字、字母、下横线
区分大小写
②、数据类型
Number：数字类型，可以用于运算
String：字符串类型，必须用引号包裹
Boolean：布尔类型，true/false
undefined：未定义 null 空类型； array 数组 ；object 对象；
何时产生？——申明了变量，但是未赋值，程序自动赋值为undefined
…
数据类型的判定：typeof
③、数据类型的转换
转String：String(需要转换的内容);
转Number：Number(需要转换的内容);
不能转换为Number的就转换为NaN
NaN：not a number（不是数字的数字类型）
NaN和任何东西做算术运算的结果都是NaN

4.算数运算符
①、加：+
坑：看是否有字符串
A、有字符串：字符串拼接
B、没有字符串：隐式转换（在这里指：把非Number转化为Number）
如：true==>1
false==>0
②、减：-（非Number类型做隐式转换）
③、乘：*（非Number类型做隐式转换）
④、除：/
⑤、取模（取余）：% 获取除不尽的部分，如：5%2 ==>1
⑥、自增：++ 或者+=
⑦、自减： – 或者 -=

5.比较运算符
①、相等： == （做隐式转换）
②、绝对相等：=== （看值且看数据类型）
③、不等：!=
④、不绝对相等：!==
⑤、大于：>
⑥、小于：<
⑦、大于等于：>=
⑧、小于等于：<=
6.逻辑运算符
①、或：|| 两个条件满足其一即可返回true
条件1||条件2
如果条件1满足，则直接返回true，不会继续判断条件2
如果条件1不满足，再继续判断条件2，条件2满足，返回true
②、且：&& 两个条件必须同时满足才能返回true
条件1&&条件2
如果条件1不满足，直接返回false，不会继续读取条件2
如果条件1满足，继续判断条件2，条件2满足，则返回true
③、非：! 取反

7.if语句
if(条件1){
条件1成立时需要做的事
}else if（条件2）{
条件2成立时需要做的事
}else{
条件1和条件2都不满足时需要做的事
}
ps：如果条件1满足，则不会继续判断后续条件

8.循环：while/for
何时使用？重复做相同或者类似的事情就需要用到循环
循环的三要素：
循环变量：记录循环的初始值以及循环次数
循环增量：让循环朝着循环结束的趋势走
循环体：需要重复做的事

9.函数:function
①、函数的申明
function函数名(参数列表){
函数体
}
②、函数的调用
方法名(参数列表)；
